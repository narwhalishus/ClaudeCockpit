Release Notes

ClaudeCockpit is a personal learning artifact, not an active product.
Roadmap · Changelog · Contributing · Testing

Status

No active launch, no product roadmap. The repo is public and the claude-cockpit package is published to npm so npx claude-cockpit works for anyone curious. There is no promotion, no launch campaign, and no support commitment.

The Claude Code wrapper/dashboard space is crowded — Claude Deck, PandaProg's ClaudeCockpit, Anthropic's own Claude Desktop, and several others. Competing meaningfully against a first-party incumbent on its own substrate is a structurally losing race. Rather than take a product swing, this codebase is published as a reference implementation: a working browser gateway for Claude Code sessions, with live streaming, tool approval, session resume, and cost estimation.

The implementation is MIT-licensed. Fork it, copy patterns, or ignore it — all fine.

For curious users

If you want to try it:

npx claude-cockpit

Requires Node.js 22+ and the Claude Code CLI installed. Opens your browser to a local cockpit that reads your existing ~/.claude/projects/ session files.

Reporting issues

GitHub Issues and Discussions are enabled. Responses are best-effort and may be slow. For anything urgent, fork and fix.

What's in the box (v1.0)

See the changelog for the full history. The short version: three-panel session cockpit, live streaming chat with markdown + tool approval, auto-generated session titles via Haiku, Bedrock cost estimation, dark and light themes, 362 tests.

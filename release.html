Hacker News
Show HN: ClaudeCockpit – Visual dashboard for Claude Code sessions

ClaudeCockpit is a browser-based dashboard for browsing and
interacting with Claude Code sessions. It reads the JSONL session
files that Claude Code writes to ~/.claude/projects/ and provides:

- Session browsing with search, pinning, and auto-generated titles
- Live streaming chat via spawned claude -p processes
- Interactive tool approval (approve/deny Bash, Edit, Write, etc.)
- Token usage and estimated Bedrock cost tracking
- One-click session summaries

Stack: Lit web components, Vite, TypeScript. Node.js gateway with
HTTP + WebSocket (custom req/res/event frame protocol). 345 tests.

Personal tool, built for a single power user on macOS, but should
work anywhere Claude Code runs.
Reddit (r/ClaudeAI)

Title: I built a visual cockpit for Claude Code — browse sessions, stream live chat, approve tools, track costs

Body: After using Claude Code heavily for a few months, I wanted a way to browse my session history, re-read old conversations, and have a visual interface for live chat with tool approval. So I built ClaudeCockpit.

It reads the JSONL files Claude Code writes to disk, so there's no API key or account needed — it just works with your existing sessions. The gateway spawns claude -p subprocesses for live chat.

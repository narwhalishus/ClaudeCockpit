v0.7.0 Live Stats & JSONL Recovery Latest
2026-04-02
Fixed
  • JSONL parser now recovers concatenated JSON objects — Claude Code sometimes omits trailing newlines, gluing two entries onto one line. New splitConcatenatedJson() uses brace-depth tracking to split them
  • renameSession appends trailing newline to prevent creating the same concatenation issue
  • Changelog inline <code> elements no longer cause extra vertical spacing (removed border, tightened padding)
Added
  • Est. Cost card updates live — app shell subscribes to chat.close events, triggering a stats re-fetch when sessions complete
  • 7 new tests for splitConcatenatedJson (nested objects, escaped quotes, string braces, malformed input)
Changed
  • Test count: 249 → 256
v0.6.0 Codebase Refinement Phase 2.5
2026-04-01
Fixed
  • Added .on("error") handlers for stdout, stderr, and stdin streams in ClaudeProcess — prevents uncaught exceptions from crashing the gateway
  • Fixed temporal dead zone in generateTitle — proc was referenced in timeout callback before its declaration
  • Gateway now aborts running claude -p processes when WebSocket client disconnects (previously only auto-denied pending approvals)
  • Abort kill timer (SIGKILL fallback) now cleaned up when process exits naturally, preventing leaked timers
Added
  • Frame protocol tests (tests/gateway/frames.test.ts) — parseFrame, serializeFrame round-trips, helper constructors
  • Server tests for sessions.summarize and chat.send WS methods
  • Settings component tests (tests/ui/settings.test.ts) — dropdown rendering, localStorage persistence
  • Shared test helper (tests/helpers.ts) — extracted renderEl and setProps from 4 duplicate locations
Changed
  • Message and chat IDs use crypto.randomUUID() instead of Date.now() — collision-free
  • MODEL_OPTIONS extracted to shared ui/constants.ts (was duplicated in chat.ts and settings.ts)
  • formatRelativeTime consolidated — single function with verbose param replaces two near-identical functions
  • Extracted getProjectDirs() helper in session-store (was duplicated in listSessions and findSessionFile)
  • Removed gatewayStartedAt: "" placeholder from computeOverviewStats — server already adds it
  • wss initialized as WebSocketServer | null for type safety
  • Pinned sessions localStorage key extracted to PINNED_SESSIONS_KEY constant
  • Test count: 189 → 213 across 13 files
